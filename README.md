Ray Tracing in One Weekend
====
*(Python edition)*


Original source:
----
https://raytracing.github.io/books/RayTracingInOneWeekend.html

<br />

Instructions
====

Setup:
> pip3 install tqdm

<br />

To run the ray tracer (on Mac),
> ./run.sh

<br />

To run the tests,
> chdir lib && ./run_tests.sh && popd
